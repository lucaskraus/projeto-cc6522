Autenticar novos usuários

UMLUseCase
Atividade 01 :: Model :: Autenticar novos usuários

Description

IDENTIFICAÇÃO: UC_10

FUNÇÃO: Autenticar Novos Usuários ATORES: Corretora, Cliente PRIORIDADE: Essencial PRÉ-CONDIÇÃO: O cliente deve ter se registrado previamente. PÓS-CONDIÇÃO: O cliente terá acesso liberado a plataforma.

• Cliente envia o cadastro preenchendo os dados próprios. • Corretora recebe os dados do cadastro juntamente as fotos. • Corretora avalia a documentação. • Corretora analisa a foto enviada pelo Cliente. • Corretora aprova o pré-cadastro. • Cliente recebe por e-mail confirmação de cadastro. • Cliente acessa a plataforma. [FS001]

[FS001] • Cliente digita a senha errada três vezes sucessivamente. • A conta do Cliente é bloqueada.
